# Прогнозирование статуса эффективности оборудования с использованием машинного обучения

Данный проект представляет собой EDA и построение моделей машинного обучения для прогнозирования класса эффективности оборудования.

## Цель
Разработка и сравнение алгоритмов машинного обучения для точного предсказания класса эффективности.

## Данные
Анализ производственных данных, содержащих технологические параметры и целевую переменную — класс эффективности.

## Технологии
* Python, Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* Модели:  Случайный лес, Градиентный бустинг, Логистическая регрессия

## Основные этапы
1. Загрузка и первичный анализ данных
2. Предобработка данных
3. Разведочный анализ (EDA)
4. Обучение моделей
5. Оценка и сравнение моделей

## Результаты
Сравнение производительности моделей.
